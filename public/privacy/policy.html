Privacy Policy

Last Updated: March 2026

1. Overview

This application is a fitness tracking tool designed to allow users to log workouts, review progress, and track fitness trends over time. This Privacy Policy explains what information is collected, how it is used, and how it is protected. By using this application, you agree to the collection and use of information in accordance with this policy.

2. Information Collected

The application collects limited information necessary to provide the fitness tracking functionality.

Account Information

When a user creates or signs in to an account, the following information may be collected:

  • Name
  • Email address
  • Team or group affiliation

Workout and Activity Data

Users may choose to log workout-related information, including:

  • Workout type
  • Exercises performed
  • Cardio activity
  • Workout duration
  • Weight or body measurements
  • Progress reflections or notes

This information is stored only for the purpose of tracking fitness activity and progress.

3. How Information Is Used

User information is used solely to support the functionality of the fitness tracking system. This includes:

  • authenticating users
  • displaying personal workout history
  • calculating progress summaries
  • generating weekly and monthly progress insights
  • supporting team dashboards for authorized administrators

The application does not sell, rent, or share personal information with third parties for marketing purposes.

4. Data Storage

User data is stored using Supabase, a managed backend platform that provides authentication services and a PostgreSQL database. Supabase securely stores account information, workout logs, progress records, and associated profile data. Access to stored data is controlled through authentication and database security policies to ensure users can only access their own data or authorized team data.

5. Data Security

Reasonable technical measures are used to protect user data. These include secure authentication through Supabase, encrypted communication over HTTPS, database access controls using row-level security policies, and restricted access to administrative functionality. While efforts are made to protect data, no system can guarantee absolute security.

6. User Access and Control

Users may view and update their own profile information and logged workout data within the application. If a user wishes to remove their data or delete their account, they may request deletion through the application administrator.

7. Third-Party Services

This application uses third-party infrastructure providers to operate the service, including Supabase for authentication and database services and a hosting platform such as Vercel for application deployment and delivery. These providers may process data as necessary to operate the application but do not have permission to use the data for their own purposes.

8. Health and Fitness Disclaimer

This application is intended only for personal fitness tracking and progress monitoring. The application does not provide medical advice and is not intended to diagnose, treat, cure, or prevent any medical condition. Users should consult qualified healthcare professionals for medical or health-related concerns.

9. Changes to This Policy

This Privacy Policy may be updated from time to time to reflect improvements or changes to the application. When updates occur, the revised version will be posted with an updated effective date.

10. Contact

For questions regarding this Privacy Policy or requests related to personal data, users may contact the application administrator.
